Realization of \mbox{\hyperlink{class_program}{Program}} class This class is intended to store info about a specefic \mbox{\hyperlink{class_program}{Program}} of studying. It has Subjects and a name. It also has a variable to indicate the path to the file where the subjects will be stored.
